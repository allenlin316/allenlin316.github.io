Ch11-Data Structure

Array

  • memory layout (計算記憶體的位址) (row-major)
    • 要算的位址 = 初始位址 + 行數 * (i-1) + (j-1)
  • 陣列適合需要大量搜尋與取值，不適合insert、delete很多的

Record

  • A record is a collection of related elements, possibly of different types, having a single

    name.

  • field 是 record 的 smallest element
  • 有點類似struct的概念

  • Array v.s. Record
    • An array defines a combination of elements, while a record defines the identifiable parts of an element.
    • e.g. : array define a class of 40 students, while record define 一個學生的不同屬性( id、name, grade...)
    • Array 是 record 的 special case，每個element是一個record with only a single field

Linked list (dynamic data structure)

  • 在鏈結串列中的每一個 node 為有兩個 field 的 record
  • 指標變數代表串列中的第一個 element，即串列的名字等同於指標變數的名字
  • 每個 node 沒有明確的名字，The name of the node is related to the name of the pointer that points to the node
  • searching a linked list
    • 用pre(previous)、cur(current)
  • deletion 比 insertion 簡單，只有兩種情況
    1. 刪除第一個 node
    1. 刪除中間或最後一個 node ，
  • Retrieving (取值)
    • 只用到cur (current)指標
    • 比 insertion、deletion 都還簡單
  • linked list 不能用 smart pointer
