    • VLIW (very long instruction word)
      • 以instruction 為單位，並在 compile 階段決定執行順序
      • 用控制器讓多個 CPU 達平行運算
      • VLIW processor 同時執行多個指令
      • VLIW 處理器相比其他多數的 超純量處理器 提供了更加強大的處理能力但是更少的硬體複雜性，但 編譯器 的複雜性提高

      • call by reference : call by reference 是 C++ 才有的功能，它對變數產生的作用與 call by address 類似，call by address 的指標內容為指向的位址，該指標仍然有記憶體位置，但是 call by reference 不會額外產生記憶體位址來指向某個變數，可以把它當作某個變數的別名
